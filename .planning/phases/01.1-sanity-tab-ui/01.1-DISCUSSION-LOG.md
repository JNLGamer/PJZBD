- Refresh model (per-frame `:render()`, no event subscription needed)

- Player handle pattern (`getSpecificPlayer(self.playerNum)`)
- File naming convention (numeric-prefix, alphabetical load order)
- Hardcoded English strings (deferred i18n)
- `tabTotalWidth` adjustment (compute new tab's text width and add it)
