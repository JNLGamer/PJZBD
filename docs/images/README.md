# Image assets for the project README

Drop image files here and reference them from the root `README.md`.

Expected files (referenced by the README):

- `cover.png` — wide hero banner at the top of the README
- `hospital.png` — mid-README atmospheric piece
- `overgrown-city.png` — wide closing banner at the bottom

Recommended aspect ratio for hero / closing banners: ~12:5 to ~16:9.
Mid-README pieces work best around 16:9 to 16:10.

If you replace any image, keep the filename so the README references stay
valid — or update the `<img>` and `![](...)` lines in `README.md` accordingly.
